# Competition Timeline

| Date | Event |
|---|---|
| **January 6, 2025** | Competition opens — development phase begins |
| **January 6 – March 28, 2025** | Development phase: submit against 2016–2019 test set |
| **March 28, 2025** | Development phase closes |
| **March 28, 2025** | Final scoring: private test set (2020–2025) is revealed |
| **April 4, 2025** | Final leaderboard published — winners announced |

## Submission limits

- **100 submissions** total per team during the development phase.
- **5 submissions per day** maximum.
- Each submission runs for at most **5 minutes** on the platform.

## Rules reminder

- Only `submission.py` is evaluated — do not rely on external internet access at inference time.
- Teams may have up to 3 members.
- Code sharing between teams before the competition ends is not allowed.
