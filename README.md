> **DataCamp Challenge · CentraleSupélec / HI-Paris**

## Task

Predict the **quarter-on-quarter GDP growth rate** (%) for 9 Euro Area countries  
(AT, BE, DE, EL, ES, IE, IT, NL, PT) using monthly leading economic indicators  
observed during the quarter.

- **Type:** Regression
- **Metric:** RMSE (lower is better)
- **Features:** BCI, CCI, SHIX, inflation, unemployment, interest rates, REER  
  (3 monthly observations per quarter × 7 indicators = 21 numeric features + country/date)
- **Target:** `GDP_growth` — QoQ % change in real GDP
- **Train period:** 2000 Q2 – 2015 Q4 (567 samples)
- **Test period:** 2016 Q1 – 2019 Q4 (144 samples, dev-phase leaderboard)
- **Private test:** 2020 Q1 – 2025 Q3 (204 samples, includes COVID-19 shock)

## Quick Start

### 1. Generate data

```bash
python tools/setup_data.py --data-dir /path/to/xlsx/files
```

### 2. Run the pipeline locally

```bash
python ingestion_program/ingestion.py \
    --data-dir    dev_phase/input_data \
    --output-dir  ingestion_res \
    --submission-dir solution

python scoring_program/scoring.py \
    --reference-dir dev_phase/reference_data \
    --prediction-dir ingestion_res \
    --output-dir scoring_res

cat scoring_res/scores.json
```

### 3. Create the Codabench bundle

```bash
python tools/create_bundle.py
# → bundle.zip
```

Upload `bundle.zip` on Codabench → Create Competition → Upload bundle.

## Baseline Results

| Split | RMSE | MAE | R² |
|---|---|---|---|
| test (2016–2019) | 1.32 | 0.81 | –0.54 |
| private_test (2020–2025) | 3.97 | 2.07 | –0.03 |

The private test set is dominated by the COVID-19 shock in 2020 Q1–Q2, where GDP dropped by up to −18%. Models robust to structural breaks will score significantly better.
